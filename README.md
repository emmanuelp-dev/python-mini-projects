A collection of beginner Python console applications including a Password Strength Checker and an Expense Tracker.

# Python Mini Projects

This repository contains beginner-friendly Python console applications built to practice core programming concepts.

## Projects

### 1. Password Strength Checker
A program that evaluates password strength based on:
- Length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

The program provides feedback and indicates whether the password meets all requirements.

### 2. Expense Tracker
A console-based application that:
- Records expenses
- Stores amount, category, note, and date
- Allows multiple entries
- Calculates and displays total expenses

## Concepts Used
- Variables
- Lists
- Dictionaries
- Loops
- Conditional Statements
- User Input
- Functions
- String Formatting

## Language
Python 3
